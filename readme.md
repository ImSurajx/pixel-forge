## 🎯 About the Project

PixelForge is a browser-based image editor built using **HTML Canvas** and **Vanilla JavaScript**.

The goal of this project is to understand how image processing works at a deeper level by manipulating pixels directly and handling real-time rendering without using external libraries.

## 🧠 Learning Focus

This project is built with a **logic-first approach**:

- Understanding **canvas rendering lifecycle**
- Working with **pixel-level data manipulation**
- Managing **state vs UI updates**
- Implementing **real-time image transformations**
- Avoiding libraries to strengthen core JavaScript fundamentals
